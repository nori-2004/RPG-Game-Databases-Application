# team_30 RPG Game Relational Database Test

## Project Summary

This project models a relational database for a role-playing video game. It represents the backend system that manages player progression, achievements, and in-game world data. The database focuses on four key areas:

- Users and their save files
- Achievements and progression rewards
- Game world, objectives, and items
- Characters and their roles

## Functionality

The database supports:

- Storing user progress and checkpoint data
- Tracking completed objectives and achievements
- Managing in-game locations, items, and characters

This model can power modern RPGs while integrating with live services (for e.g., PlayStation accounts) to sync user achievements and progress.
